### spring ioc 容器初始化

#### 总体分3步定位，加载，注册

+ 定位：定位配置文件和扫描相关的注解
+ 加载：将配置信息加载到内存中
+ 注册：根据载入的信息将对象初始化到IOC容器中

寻找入口 -> 获取配置文件路径 -> 容器开始启动 -> 开始解析配置文件 -> 载入配置路径 -> 分配路径处理策略 -> 解析配置文件路径 -> 加载配置文件信息 -> 开始解析配置文件

 分配解析配置文件策略 -> 将配置文件载入到内存 -> 载入<bean> 元素 -> 载入<property> 元素 -> 载入<property> 子元素 -> 载入<list> 元素

分配注册策略 -> 开始向容器注册
